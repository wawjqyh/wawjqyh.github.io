# 帮助命令

## man

即manual，帮助文档

### 选项

* `-a` 在所有的man帮助手册中搜索
* `-f` 等价于whatis指令，显示给定关键字的简短描述信息
* `-P` 指定内容时使用分页程序
* `-M` 指定man手册搜索的路径

```javascript
man ls
man mkdir

//只会显示sleep命令的手册,如果想查看库函数sleep
man 3 sleep
```

#### 级别

* `1` 查看命令的帮助
* `2` 查看可被内核调用的函数的帮助
* `3` 查看函数和函数库的帮助
* `4` 查看特殊文件的帮助（主要是/dev目录下的文件）
* `5` 查看配置文件的帮助
* `6` 查看游戏的帮助
* `7` 查看其它杂项的帮助
* `8` 查看系统管理员可用命令的帮助
* `9` 查看和内核相关文件的帮助
