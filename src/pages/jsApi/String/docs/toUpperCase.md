# String.prototype.toUpperCase()

toUpperCase() 将调用该方法的字符串值转换为大写形式，并返回。

## 语法

```javascript
str.toUpperCase()
```

## 描述

toUpperCase 将调用该方法的字符串值转换为大写形式，并返回。toUpperCase 方法不影响字符串本身的值。

## 示例

```javascript
​console.log( "alphabet".toUpperCase() ); // "ALPHABET"
```
