# Number.NaN

## 概述

Number.NaN 表示“非数字”（Not-A-Number）。和 NaN 相同。

不必创建一个 Number 实例来访问该属性，使用 Number.NaN 来访问该静态属性。

Number.NaN 属性的属性特性：

|             |       |
|:------------|:------|
|writable     | false |
|enumerable   | false |
|configurable | false |
