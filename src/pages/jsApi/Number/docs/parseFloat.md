# Number.parseFloat()

## 概述

Number.parseFloat() 方法可以把一个字符串解析成浮点数。该方法与全局的 parseFloat() 函数相同，并且处于 ECMAScript 6 规范中
（用于全局变量的模块化）。

## 语法

```javascript
Number.parseFloat(string)
```

### 参数

* `string` 被解析的字符串

## 描述

请移步全局函数 parseFloat() 页面查看更多的解释和示例。
